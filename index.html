ntd - not std

ntd (acronym: "​​n​ot s​​td​​") is a library which adds functionality to C++ which should be included in the standard lib by default (IMHO).

Note: This library is using some shiny new C++14 features, so you'll need a C++14 compatible compiler.

Install

homebrew (OSX)

brew install https://raw.githubusercontent.com/kasoki/ntd/master/misc/ntd.rb

Dependencies

None, at least if you're only using the library. If you plan to compile the unit tests you'll need the unit test framework: mavis.

What stuff does ntd contain?

  • curry, ntd/curry.hpp
  • string tools, ntd/string.hpp
  • memory, ntd/memory.hpp
